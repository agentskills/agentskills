# Quickstart: Your First Composed Skill

**Goal:** Run a working skill composition in under 10 minutes. Theory comes later.

> **Why this architecture?** AgentSkills follows [12-Factor Agents](https://github.com/humanlayer/12-factor-agents) principlesâ€”the same thinking that transformed web services applies to AI agents. Instead of monolithic mega-prompts, we build **small, composable skills** with **strong contracts** and **clear safety boundaries**. Skills are portable across runtimes: the same definition works whether you're running locally, in Claude Code, or enterprise infrastructure.

**Look at the composition:**

```yaml
# examples/quickstart/weather-report/SKILL.md
name: weather-report
level: 2                    # L2 = composite skill
operation: READ             # Safe, no confirmation needed

composes:
  - fetch-weather           # L1: Get raw weather data
  - format-report           # L1: Format into human-readable

inputs:
  - name: location
    type: string
    required: true

outputs:
  - name: report
    type: string
```

**That's it.** Two skills composed with 4 lines of YAML.

## 5-8 Minutes: Create Your Own Skill

Create a new L1 skill:

```bash
mkdir -p examples/quickstart/my-skill
```

Create `examples/quickstart/my-skill/SKILL.md`:

```yaml
name: greet-user
level: 1
operation: READ

description: |
  Generates a personalized greeting.

inputs:
  - name: username
    type: string
    required: true
  - name: formal
    type: boolean
    default: false

outputs:
  - name: greeting
    type: string
```
