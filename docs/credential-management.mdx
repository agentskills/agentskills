# Credential Management for Skills

This document describes the credential system for skills, adapted from n8n's production-grade architecture.

## Overview

Skills that interact with external services need credentials. The credential system provides:

- **Secure storage** - AES-256-CBC encryption at rest
- **OAuth2 flows** - Authorization code, client credentials, PKCE
- **Auto-refresh** - Transparent token refresh on 401
- **Permission model** - User, project, and global scopes
- **Type inheritance** - Base types extended by service-specific configs

## Credential Types

### Base Types

| Type | Use Case | Fields |
|------|----------|--------|
| `oauth2` | OAuth2 authorization code | client_id, client_secret, auth_url, token_url |
| `oauth2-client-credentials` | Machine-to-machine | client_id, client_secret |
| `api-key` | Simple API key | api_key, header_name |
| `basic-auth` | Username/password | username, password |
| `bearer-token` | Static bearer token | token |
| `custom` | Custom authentication | user-defined fields |

## Storage & Encryption

### Encryption at Rest

All credential data is encrypted using AES-256-CBC:

```python
from cryptography.hazmat.primitives.ciphers import Cipher, algorithms, modes
from cryptography.hazmat.backends import default_backend
import os
import hashlib

class CredentialCipher:
    """AES-256-CBC encryption for credential storage."""

    SALT_PREFIX = b"Salted__"

    def __init__(self, encryption_key: str):
        self.encryption_key = encryption_key.encode()

    def encrypt(self, data: dict) -> str:
        """Encrypt credential data."""
        salt = os.urandom(8)
        key, iv = self._derive_key_iv(salt)

        cipher = Cipher(algorithms.AES(key), modes.CBC(iv), backend=default_backend())
        encryptor = cipher.encryptor()

        plaintext = json.dumps(data).encode()
        # PKCS7 padding
        padding_length = 16 - (len(plaintext) % 16)
        padded = plaintext + bytes([padding_length] * padding_length)

        encrypted = encryptor.update(padded) + encryptor.finalize()
        return base64.b64encode(self.SALT_PREFIX + salt + encrypted).decode()

    def decrypt(self, encrypted_data: str) -> dict:
        """Decrypt credential data."""
        raw = base64.b64decode(encrypted_data)
        salt = raw[8:16]
        ciphertext = raw[16:]

        key, iv = self._derive_key_iv(salt)

        cipher = Cipher(algorithms.AES(key), modes.CBC(iv), backend=default_backend())
        decryptor = cipher.decryptor()

        padded = decryptor.update(ciphertext) + decryptor.finalize()
        # Remove PKCS7 padding
        padding_length = padded[-1]
        plaintext = padded[:-padding_length]

        return json.loads(plaintext.decode())

    def _derive_key_iv(self, salt: bytes) -> tuple[bytes, bytes]:
        """EVP_BytesToKey-compatible key derivation."""
        password = self.encryption_key + salt

        hash1 = hashlib.md5(password).digest()
        hash2 = hashlib.md5(hash1 + password).digest()
        hash3 = hashlib.md5(hash2 + password).digest()

        key = hash1 + hash2  # 256 bits
        iv = hash3  # 128 bits

        return key, iv
```

### Database Schema

```sql
-- Credentials table
CREATE TABLE credentials (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    name VARCHAR(128) NOT NULL,
    type VARCHAR(128) NOT NULL,  -- e.g., 'slack-oauth2'
    data TEXT NOT NULL,          -- Encrypted JSON blob
    is_managed BOOLEAN DEFAULT FALSE,
    is_global BOOLEAN DEFAULT FALSE,
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
);

-- Credential sharing (many-to-many with projects)
CREATE TABLE shared_credentials (
    credential_id UUID REFERENCES credentials(id),
    project_id UUID REFERENCES projects(id),
    role VARCHAR(32) NOT NULL,   -- 'owner' | 'user'
    PRIMARY KEY (credential_id, project_id)
);

-- Indexes
CREATE INDEX idx_credentials_type ON credentials(type);
CREATE INDEX idx_shared_credentials_project ON shared_credentials(project_id);
```

## OAuth2 Token Management

### Token Lifecycle

```
1. User initiates OAuth2 flow
   ↓
2. Authorization URL opened in browser
   ↓
3. User grants permissions
   ↓
4. Callback receives authorization code
   ↓
5. Exchange code for tokens
   ↓
6. Store encrypted tokens
   ↓
7. Use access_token for API calls
   ↓
8. On 401: refresh token automatically
   ↓
9. Store new tokens
```

### Automatic Token Refresh

```python
class OAuth2TokenManager:
    """Manages OAuth2 token lifecycle with automatic refresh."""

    async def execute_with_auth(
        self,
        credential_id: str,
        request_fn: Callable
    ) -> Any:
        """Execute request with automatic token refresh."""
        credential = await self.get_credential(credential_id)
        token_data = credential.get_oauth_tokens()

        try:
            return await request_fn(token_data.access_token)

        except HTTPError as e:
            if e.status_code == 401:
                # Token expired, refresh it
                new_tokens = await self.refresh_token(
                    credential,
                    token_data.refresh_token
                )

                # Persist new tokens
                await self.update_tokens(credential_id, new_tokens)

                # Retry with new token
                return await request_fn(new_tokens.access_token)

            raise

    async def refresh_token(
        self,
        credential: Credential,
        refresh_token: str
    ) -> TokenData:
        """Exchange refresh token for new access token."""
        cred_type = self.get_credential_type(credential.type)

        response = await self.http_client.post(
            cred_type.oauth2.token_url,
            data={
                "grant_type": "refresh_token",
                "refresh_token": refresh_token,
                "client_id": credential.data.client_id,
                "client_secret": credential.data.client_secret,
            }
        )

        return TokenData(
            access_token=response["access_token"],
            refresh_token=response.get("refresh_token", refresh_token),
            expires_at=time.time() + response["expires_in"],
            token_type=response.get("token_type", "Bearer"),
        )
```

## Permission Model

### Scope Hierarchy

```
Global Scopes (Admin)
├── credential:list        - List all credentials
├── credential:create      - Create any credential
├── credential:delete      - Delete any credential
└── credential:share-global - Make credentials global

Project Scopes
├── credential:read        - View credential metadata
├── credential:update      - Modify credential
└── credential:use         - Use credential in skills

Role Mappings
├── owner  → [read, update, delete, share]
├── editor → [read, update, use]
└── viewer → [read, use]
```

### Access Control

```python
class CredentialAccessControl:
    """Controls access to credentials based on user permissions."""

    async def check_access(
        self,
        user: User,
        credential_id: str,
        required_scope: str
    ) -> bool:
        """Check if user can perform action on credential."""

        # Global admin check
        if self.has_global_scope(user, required_scope):
            return True

        # Check project-level permissions
        shared = await self.get_shared_credential(credential_id, user.project_ids)
        if not shared:
            return False

        # Check role grants required scope
        return self.role_grants_scope(shared.role, required_scope)

    async def get_credentials_for_user(
        self,
        user: User,
        credential_type: str = None,
        include_global: bool = True
    ) -> list[Credential]:
        """Get all credentials user can access."""

        credentials = []

        # Get credentials shared with user's projects
        project_creds = await self.get_project_credentials(
            user.project_ids,
            credential_type
        )
        credentials.extend(project_creds)

        # Include global credentials if requested
        if include_global:
            global_creds = await self.get_global_credentials(credential_type)
            credentials.extend(global_creds)

        return credentials
```

## Credential Resolution at Runtime

### Skill Execution Flow

```python
class SkillExecutor:
    """Executes skills with credential resolution."""

    async def execute(
        self,
        skill_name: str,
        inputs: dict,
        user: User
    ) -> dict:
        """Execute skill with resolved credentials."""

        skill = await self.load_skill(skill_name)

        # Resolve all required credentials
        resolved_credentials = {}
        for cred_req in skill.credentials:
            credential = await self.resolve_credential(
                user=user,
                credential_type=cred_req.type,
                required_scopes=cred_req.scopes,
                alternatives=cred_req.alternatives
            )

            if credential is None and cred_req.required:
                raise CredentialNotFoundError(
                    f"No {cred_req.type} credential available"
                )

            alias = cred_req.alias or cred_req.type
            resolved_credentials[alias] = credential

        # Execute skill with credentials
        return await skill.execute(
            inputs=inputs,
            credentials=resolved_credentials,
            token_manager=self.token_manager
        )

    async def resolve_credential(
        self,
        user: User,
        credential_type: str,
        required_scopes: list[str],
        alternatives: list[str] = None
    ) -> Credential | None:
        """Resolve credential for user, trying alternatives if needed."""

        # Try primary type first
        credential = await self.find_credential(user, credential_type)
        if credential and self.has_required_scopes(credential, required_scopes):
            return credential

        # Try alternatives
        for alt_type in (alternatives or []):
            credential = await self.find_credential(user, alt_type)
            if credential:
                return credential

        return None
```

```typescript
// Frontend: Initiate OAuth2 connection
async function connectOAuth2(credentialType: string) {
  // 1. Get authorization URL from backend
  const { authUrl, state } = await api.post('/credentials/oauth2/init', {
    type: credentialType,
    callbackUrl: `${window.location.origin}/credentials/callback`
  });

  // 2. Open popup for authorization
  const popup = window.open(authUrl, 'oauth2', 'width=600,height=700');

  // 3. Listen for callback
  window.addEventListener('message', async (event) => {
    if (event.data.type === 'oauth2-callback') {
      // 4. Exchange code for tokens (backend)
      const credential = await api.post('/credentials/oauth2/callback', {
        code: event.data.code,
        state: event.data.state
      });

      popup.close();
      onCredentialConnected(credential);
    }
  });
}
```

## Security Considerations

### Never Expose

- Access tokens to frontend (except during OAuth callback)
- Refresh tokens (ever)
- Encryption keys
- Client secrets (mask in UI)

### Always Encrypt

- All credential data at rest
- Tokens in transit (HTTPS only)
- Backup/export data

## Migration from Existing Credentials

### Import Credentials

```python
async def import_credential(
    self,
    credential_type: str,
    data: dict,
    user: User
) -> Credential:
    """Import credential from external source."""

    # Validate against credential type schema
    cred_type = self.get_credential_type(credential_type)
    validated = cred_type.validate(data)

    # Encrypt and store
    encrypted = self.cipher.encrypt(validated)

    return await self.create_credential(
        name=data.get('name', f'Imported {credential_type}'),
        type=credential_type,
        data=encrypted,
        user=user
    )
```

### Export Credentials (Admin Only)

```python
async def export_credentials(
    self,
    user: User,
    include_tokens: bool = False
) -> list[dict]:
    """Export credentials for backup (admin only)."""

    if not self.has_global_scope(user, 'credential:export'):
        raise PermissionError("Export requires admin scope")

    credentials = await self.get_all_credentials()

    return [
        {
            'name': c.name,
            'type': c.type,
            'data': self.cipher.decrypt(c.data) if include_tokens else {
                k: v for k, v in self.cipher.decrypt(c.data).items()
                if k not in ['access_token', 'refresh_token']
            }
        }
        for c in credentials
    ]
```

### 3. Skill Implementation

```python
async def execute(
    self,
    inputs: dict,
    credentials: dict,
    token_manager: TokenManager
) -> dict:
    """Post message to Slack channel."""

    slack_cred = credentials['slack-oauth2']

    async with token_manager.auth_context(slack_cred) as auth:
        response = await self.http.post(
            "https://slack.com/api/chat.postMessage",
            headers={"Authorization": f"Bearer {auth.access_token}"},
            json={
                "channel": inputs["channel"],
                "text": inputs["message"]
            }
        )

    if not response["ok"]:
        raise SkillExecutionError(response["error"])

    return {
        "ts": response["ts"],
        "channel": response["channel"]
    }
```
